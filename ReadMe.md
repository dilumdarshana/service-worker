# Web Worker using Vanilla JS

## Technologies used
- ECMASCRIPTS
- NodeJS
- ReactJS
- Web Sockets

## Why service worker?
- CPU extensive task can be run in separate worker thread without distrupting the main thread

## Limitations
- Only runs on https (unless localhost)
- Can not access DOM
